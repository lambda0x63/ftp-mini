## [0.4.0] - 2025-05-28

### 추가
- SFTP 프로토콜 지원 추가 (#4)
  - 설정 마법사에서 FTP/SFTP 프로토콜 선택 가능
  - SSH2 라이브러리를 사용한 SFTP 구현
  - 기존 FTP 기능과 동일한 파일 작업 지원
- FTP 포트 설정 기능 추가 (#2)
  - 사용자 정의 포트 번호 설정 가능
  - 포트 유효성 검사 (1-65535)
  - 기본 포트 자동 설정 (FTP: 21, SFTP: 22)

### 개선
- 크로스 플랫폼 경로 처리 개선
- 연결 및 재연결 안정성 향상
- 에러 처리 및 로깅 시스템 강화
- 파일 동기화 정확성 개선

### 기술적 변경사항
- SFTPManager 클래스 추가
- FTPManager에 프로토콜 선택 로직 통합
- 포트 설정 UI 및 유효성 검사 구현
- 경로 동기화 문제 해결

### 호환성
- 이전 버전과 완전 호환
- 기존 FTP 설정에 영향 없음
